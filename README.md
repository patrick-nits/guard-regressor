# Guard::Regressor
Guard Plugin for https://github.com/ndea/regressor

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'guard-regressor'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install guard-regressor

## Usage

    $ guard init regressor
